# Старооскольский гид — сайт-гид по городу Старый Оскол

Веб-приложение на Flask + SQLite, которое позволяет жителям и гостям города узнавать о достопримечательностях, оставлять отзывы, ставить оценки. Администратор может добавлять, редактировать и удалять места.

## 🚀 Функционал

- Просмотр списка достопримечательностей с фильтром по категориям
- Карточка каждого места: описание, адрес, фото, рейтинг, количество просмотров
- Авторизация и регистрация пользователей
- Добавление отзывов и оценок (только для авторизованных)
- Админ-панель: CRUD для мест
- Современный адаптивный дизайн (Bootstrap 5 + кастомные стили)

## 🛠️ Технологии

- Python 3.9+
- Flask 2.3
- SQLite3
- Bootstrap 5
- HTML5 / CSS3 / JavaScript
